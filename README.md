
#### MacPrepE - (Mac Preparation Easy)
Install or update all tools from my GitHub for macOS.
- eping(a),dscp-top,muxpi,mau-tg
  
Includes automatic installation of Homebrew and additional brew‑based utilities.   
- (joe,python,uv,pipx,nmap,iperf,iperf3,btop,tcpdump,fping,curl,tmux,screen,tcpreplay,[pip: scapy])

```
/bin/zsh -c "$(curl -fsSL https://raw.githubusercontent.com/ewaldj/microups/refs/heads/main/MacPrepE.sh)"
```
